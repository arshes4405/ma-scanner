▶ 매수 조건 (전체 AND)
----------------------------------------------------------------
  1. 현재봉 양봉
  2. 직전봉 음봉
  3. 직전봉 RSI(14) < 티어별 임계값
  4. 현재봉 RSI < (직전봉RSI + 2 + floor(경과분/10))
  5. 현재가 < 직전봉 (고가+저가)/2
  6. 직전봉 (저가+종가)/2 < 볼린저 임계(20,2) — 티어별 fromLower
  7. 24시간 거래량 >= $1,000,000
  8. 언랭 코인은 거래량 상위 250개만 스캔

▶ 매매 설정
----------------------------------------------------------------
  [매수]
  - 알트 신규 : 티어별 금액 | 30x Isolated (실패시 20x → 10x)
  - 알트 DCA  : 다음 봉 재신호 시 초기금액 10% 추가, 티어별 최대
  - 메이저    : $10,000 | 50x Cross (실패시 20x), DCA 없음

  [SL/TP - positionMonitor]
  - 반익절 전 : -3% 전량 청산
  - +5% 도달  : 절반 매도
  - 반익절 후 : +0.5% 이하 시 나머지 청산
  - notional < $2,000 종목만 관리

  [SL 쿨다운]
  - SL 후 8시간 재매수 금지 (메이저 제외)
  - trade_log.csv의 SL/AUTO_SL 기준으로 판단

▶ 자동 티어 관리 (promoteCoins)
----------------------------------------------------------------
  - 매시 5분 자동 실행 (updateTrades 5분 후)
  - trade_log.csv 전체 분석 → tier_config.json 갱신
  - TIER 기준: TIER1(순익절>=5) / TIER2(>=3) / TIER3(>=1)
  - 블랙 기준: 순익절 <=-2 → EXCLUDE_SYMBOLS 자동 추가
  - SYSTEM SL + MANUAL AUTO_SL 2패스 집계 (중복 방지)

----------------------------------------------------------------
▶ trade_log.csv 스키마
----------------------------------------------------------------
  datetime, symbol, action, entry_price, exit_price,
  qty, pnl_pct, pnl_usdt, source, order_id

  - SYSTEM : positionMonitor 자동 (SL/TP_HALF/BE_CLOSE)
             order_id = Binance orderId
  - MANUAL : updateTrades 수집 (AUTO_CLOSE/AUTO_SL)
             order_id = SYMBOL_SEC 포맷 (예: ATUSDT_1746403354)

▶ 백테스트 결론 (2026-05-05 기준)
----------------------------------------------------------------
  [결론 1] 하락장에서 알트 매매는 하지 않는 것이 최선
    - 1~2월(하락장) 언랭 결과: 승률 ~28~30%, BIAS 관계없이 전부 손실
    - 손익분기 승률 37.5% (TP5%/SL3%)를 하락장엔 달성 불가
    - 메이저(ETH/HYPE)만 하락장에도 수익 → 메이저는 예외

  [결론 2] Adaptive BIAS (전주 수익→+5, 손실→-5)는 효과 없음
    - Adaptive -8,480 USDT vs BIAS 0 고정 -6,220 USDT (오히려 악화)
    - 1주 lag 반응이라 추세 전환 타이밍에 역방향 베팅 발생

  [결론 3] 거래량 상위 250개 제한은 노이즈 (이미 반영)
    - 251위~ 코인이 상승장에서 오히려 성과 더 좋았음
    - UNRANKED_LIMIT 제거 완료 (v44)

  ★★★ [결론 4] ETH 주봉 RSI 방향 필터 → 핵심 전략 (2026-05-05 검증)
    - 조건: 현재 주봉 RSI > 전주 주봉 RSI → 알트 매수 허용, 아니면 전량 스킵
    - 대상: 거래량 251위~ 205개, 기간: 2026-01-01 ~ 2026-04-30
    - 결과:
        필터 없음  : -6,220 USDT  (34.4% 승률, 3,442건)
        ETH RSI 필터: +6,490 USDT  (46.3% 승률,   925건)  ← +12,710 USDT 차이
    - 월별:
        1월: -530  (295건, 1주만 매수)
        2월: +0    (0건,   전주 스킵)
        3월: +1,320 (116건, 1주만 매수)
        4월: +5,700 (514건, 4주 매수)
    - 핵심: 하락장(ETH RSI 하강 중) 알트 자동 스킵 → 손실 회피 + 승률 46%로 상승
    - 메이저(ETH/HYPE)는 이 필터 적용 제외 (항상 매수)
    - 향후 과제: newFloorScaner에 적용 시 ETH 주봉 RSI 실시간 계산 필요
                 (1시간봉으로 현재 주봉 close 조합 → 주봉 RSI 계산)
